# P1 and P2 Thread Data Dump

`P1` and `P2` Thread Data Dump CSV files have headers:

- X axis: `MAXTHREADS`
- Y axis: `Time (in nanoseconds)`

Various folders represent various different matrices:

- Row and Column Matrix Multiplication
  - Matrix One: $1000 \times 1$
  - Matrix Two: $1 \times 1000$

- Square Matrix Multiplication
  - Matrix One: $500 \times 500$
  - Matrix Two: $500 \times 500$

- Skewd Matrix Multiplication
  - Matrix One: $100000 \times 67$
  - Matrix Two: $67 \times 32$
